La carpeta "Apps" contiene las funciones .mlapp para ejecutar la aplicacion en MATLAB AppDesigner. Es necesario añadir el repositorio con sus subcarpetas al path de MATLAB para la ejecución de dichos programas.

La carpeta interfaz/Glaucoma_diagnosis App exportada/for_redistribution_files_only/ contiene el ejecutable final que se descarga para la ejecucion sin MATLAB. Para ello es necesario tener instalado MATLAB R2024a o el MATLAB Runtime R2024a.
