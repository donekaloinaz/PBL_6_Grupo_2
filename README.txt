Este trabajo es un proyecto desarrollado por un grupo de trabajo de 5 integrantes de tercer curso de Ingeniería Biomédica de Mondragon Unibertsitatea.
Se ha desarrollado un prototipo de apoyo al diagnóstico de Glaucoma mediante métodos de aprendizaje automático. Este prototipo es parte de una Start-Up y es por ello que también se ha desarrollado una página web que describe el prototipo ofrecido.
Nota: este trabajo no debe ser usado de manera profesional ya que carece de de las capacidades y cumplimiento de las normativas y legislaciones necesarias.
