Este trabajo es un proyecto desarrollado por un grupo de trabajo de 5 integrantes de tercer curso de Ingeniería Biomédica de Mondragon Unibertsitatea.

Se ha desarrollado un prototipo de apoyo al diagnóstico de Glaucoma mediante métodos de aprendizaje automático.

Este prototipo es parte de una Start-Up y es por ello que también se ha desarrollado una página web que describe el prototipo ofrecido.

Nota: este trabajo no debe ser usado de manera profesional ya que carece de de las capacidades y el cumplimiento de las normativas y legislaciones necesarias. 

Los contenidos de este repositorio se describen de la siguiente manera:
- certificaciones_y_normativas: aqui se encuentra la documentación en cuanto al cumplimiento de la normativa realizada hasta la fecha
- data: aqui se encuentran todos los datos utilizados para el desarrollo del proyecto y a su vez todos los datos creados durante el proceso, como imágenes, modelos de predicción,...
- docs: aqui se encuentran todos los archivos relacionados con la página web para su visualización y configuración, junto con archivos de datos XML 
- interfaz: aqui se encuentran la aplicación y todas las funciones, archivos y datos necesarias para abrirla por MATLAB App Designer. Tambien se incluye la Aplicación exportada standalone sin el runtime de MATLAB incluido, por lo que habrá que descargarlo a parte en un ordenador sin MATLAB instalado siguiendo las instrucciones en el archivo de texto en la carpeta.
- procesado_datos: aqui se encuentran todos los códigos de MATLAB que han sido desarrollados para la creación de modelos de predicción tanto para el filtro de calidad como para el clasificador final de glaucoma
- procesado_imagenes: aqui se encuentran todos los códigos de MATLAB que han sido desarrollados para realizar el procesamiento de imagen para la extracción de características

Cualquier duda en cuanto al funcionamiento o utilización de este proyecto contactar con cualquiera de los colaboradores.
